## 认知能力：我们是如何认知的？什么在影响我们的认知？怎么提高我们的认知

人类的 🧠 是如何学习到知识的？通过不断思考。在不断地学习和思考过程中，人的大闹内部产生着剧烈的神经运动，脑神经之间建立了更多的连接。随着神经元之间连接越来越多，我们对问题的认知也就越来越深刻，遇到类似问题反映也会越来越快。不仅如此，甚至还能触类旁通，举一反三，乃至和其它领域结合，产生创新性的想法。这表现出来就是你更聪明了。
