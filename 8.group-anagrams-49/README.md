# 字母异位词分组

> 给定一个字符串数组，请将`字母异位词`组合在一起。可以按任意顺序返回结果列表。
> `字母异位词`: 是由重新排列源单词的字母得到的一个新单词，所有源单词中的字母都恰好只用一次

## 示例：
---
```
输入：str = ["eat", "tea", "tan", "ate", "nat", "bat"]
输出：[["bat"],["nat","tan"],["ate","eat","tea"]]
```
---

```
输入：str = [""]
输出：[[""]]
```

```
输入：str = ["a"]
输出：[["a"]]
```

提示：
  - 1 <= str.length <= 104
  - 0 <= str[i].length <= 100
  - str[i] 仅包含小写字母
