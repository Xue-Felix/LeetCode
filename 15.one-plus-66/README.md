# 加一
> 给定一个由`数组`组成的`非空`数组所表示的非负整数，在该书的基础上`加一`。
> 最高位数字存放在数组的首位，数组中每个元素只存储单个数字。
> 你可以假设除了整数0之外，这个证书不会以0开头

## 示例：
---
```
输入：digits = [1,2,3]
输出：[1,2,4]
解释：输入数组表示数字 123。
```
---
```
输入：digits = [4,3,2,1]
输出：[4,3,2,2]
解释：输入数组表示数字 4321。
```
---
```
输入：digits = [0]
输出：[1]
```
---

提示：
  - 1 <= digits.length <= 100
  - 0 <= digits[i] <= 9
