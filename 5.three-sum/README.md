# 三数之和

> 给你一个包含n个整数的数组`nums`,判断`nums`中是否存在三个元素a, b, c，使得a+b+c=0？请你找出所有和为0且不重复的三元组。

## 示例：

```
输入：nums = [-1, 0, 1, 2, -1, -4];
输出：[[-1,-1,2],[-1,0,1]]
```

---

```
输入：nums = [-1, 0, 1, 2, -1, -4];
输出：[]
```
---

```
输入nums = [0]
输出: []
```

提示：
  - 0 <= nums.length <= 3000
  - -10**5 <= nums[i] <= 10**5
